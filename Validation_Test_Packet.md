## Phase 1: CSV Upload & Card Grid

### 1. CSV Upload UI Modal
The user can click the "Import CSV" button from the main dashboard to open a dedicated upload modal. This modal supports drag-and-drop or clicking to select a `.csv` file. It uses Papaparse to parse standard formats and Moxfield/ManaBox exports.
